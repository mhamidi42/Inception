# Inception
Inception est un projet de system administration réalisé dans le cadre du cursus 42.   L’objectif est de créer une petite infrastructure web utilisant Docker et Docker Compose.

Le projet met en place plusieurs conteneurs interconnectés pour héberger un site WordPress, avec un serveur web NGINX, une base de données MariaDB et des volumes persistants.

## Objectif
Ce projet permet d'apprendre à :
- utiliser Docker et Docker Compose
- créer et configurer plusieurs conteneurs
- gérer un réseau entre services
- utiliser des volumes persistants
- configurer un serveur web sécurisé avec TLS

## Technologies
- Docker
- Docker Compose
- NGINX
- WordPress
- MariaDB
- Linux
